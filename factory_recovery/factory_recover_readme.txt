This folder contains the two designs loaded at the factory
--> LedBreathe -- a simple pwm light shift
--> DECA_Gsensor -- an demo design using the accelerometer, led's light up when the board is tilted, almost like a level

If you have overwritten your CFM flash memory in MAX 10, you can restore the flash by running
--> restore_deca.bat

Please note, if you have a REV B board, you will need to run:
--> restore_deca_revb.bat
